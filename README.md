#---------------- PROGRAM LOGIC ---------------#

0 - to quit
1  - to add
2 - to subtract
3 - multiply
4 - Divide
5 - modulus

- The program continuously runs in a loop, displaying a menu of operations.
- The user selects an operation by entering a number (0 - 5).
- The program asks for two numbers as input and performs the selected operation.
- If the user enters 0, the program exits.
- If an invalid command is entered, an error message is displayed.
